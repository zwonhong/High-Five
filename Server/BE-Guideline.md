### BE - Socket Communication
* 커넥션 관리 (유저 접속 시 소켓 ID와 유저 ID 매핑)
* 이벤트 핸들링 (메시지 수신, 유저 상태 변경 등)
* 브로드캐스팅 (전체 유저에게 메시지나 드로잉 전송)
* 에러 처리 (네트워크 문제, 잘못된 데이터 등)
* 멀티 룸 지원 (여러 게임 방 관리)
* 멀티 플레이어 동기화 (게임 상태, 점수 등 실시간 업데이트)
* 최적화

### BE - Game Logic
* 게임 상태 관리 (게임 시작, 종료, 턴 변경 등)
* 점수 계산 (정답 여부에 따른 점수 부여)
* 게임 룰 구현 (카드 배분, 턴 순서 등)
* 타이머 관리 (턴 제한 시간, 게임 시간 등)
* 유저 관리 (닉네임, 점수, 방 정보 등)
* MVC 패턴 디자인 (방 정보, 게임 상태, 유저 정보, 게임 룰, (예시-드로잉 힌트 등 추가가능))
